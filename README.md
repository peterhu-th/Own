# 记忆窝点 (Diary-Room)

一个私密、温馨的双人专属日记本微信小程序。专为情侣、闺蜜或挚友打造，致力于提供一个没有外界干扰、只属于两个人的静谧记录空间。

## 🌟 功能特性 (Features)

- **专属私密空间**：通过 6 位专属邀请码绑定彼此，一旦绑定成功，系统将为您和 Ta 建立起牢不可破的数据隔离墙，保证日记内容仅你们两人可见。
- **时光日记本**：
  - **发布与回溯**：支持“记录此刻”，也支持“捡拾过去”的补传功能（可精确到分钟）。
  - **富媒体记录**：支持长文本与最多 9 张图片，点击即可大图预览与长按保存。
  - **便捷管理**：支持随时修改和删除日记，日记删除时将自动同步清理相关批注，保障数据整洁。
- **互动批注**：不仅可以给自己，也可以为对方的日记写下批注，开启跨越时空的对话。
- **实时情感连接**：
  - **在线状态感知**：基于心跳机制，实时显示对方是否在线以及你们累积的陪伴时长。
  - **时光轴动态**：以精美的 Timeline 形式记录双方的每一次互动（发布日记、写下批注、绑定关系等），不漏掉任何一个心动瞬间。
- **个性化装扮**：深度接入微信原生能力，支持一键获取并上传微信头像、修改专属昵称。
- **安全防护机制**：邀请码验证失败超过 3 次自动锁定 1 分钟，防爆破，保护隐私。

## 🎨 视觉与交互体验

项目抛弃了传统的“后台管理系统”式审美，采用了极具亲和力的 UI 设计：

- **莫兰迪治愈配色**：以温暖的西柚/珊瑚橘 (`#FF7A59`) 为主色调，结合清新的莫兰迪绿 (`#8FBC8F`)，营造温暖氛围。
- **去线化与悬浮纸片感**：全局摒弃生硬的实体边框，大面积采用白底、淡粉橘色底，配合大圆角 (`20rpx`) 与极微弱的暖色弥散阴影，打造“漂浮纸片”的轻量感。
- **呼吸感排版**：充分的留白，为“写日记”和“看日记”提供如同写信一般的沉浸体验。

## 🛠 技术栈 (Tech Stack)

- **前端框架**：[uni-app](https://uniapp.dcloud.net.cn/) + **Vue 3** (Composition API) + **Vite**
- **状态管理**：**Pinia**
- **后端支撑**：**微信云开发 (WeChat Cloud Base)**
  - 云函数 (Cloud Functions)：承担所有核心业务逻辑及权限校验。
  - 云数据库 (Cloud Database)：NoSQL 灵活存储。
  - 云存储 (Cloud Storage)：存储用户上传的图片及头像。

## 📂 项目架构与云函数划分

整个项目的服务端逻辑由四个高度解耦的云函数构成，确保了权限的严格隔离：

1.  **`authCloud` (鉴权中心)**
    - 负责静默登录、初始化用户数据。
    - 负责生成 15 分钟有效期的邀请码、处理双方绑定逻辑。
    - 处理全局的心跳轮询，计算并更新双方的在线时长 (`total_online_minutes`) 和最后活跃时间。
2.  **`diaryCloud` (日记中枢)**
    - 负责日记的发布、修改、删除。
    - 负责日记列表的获取（支持分页加载加载，内置 `skip/limit`）。
    - 包含发布时间的时间合法性校验，拦截未来时间。
3.  **`commentCloud` (互动模块)**
    - 处理针对特定日记的批注增加与删除。
    - 严格校验删除操作的所有者权限。
4.  **`userCloud` (用户与动态模块)**
    - 获取个人配置、拉取伴侣实时状态。
    - 更新用户头像（`avatarUrl`）与昵称（`nickname`）。
    - 获取双方的时间轴历史动态 (`ActivityLog`)。

## 🚀 本地运行与部署

### 1. 环境准备

- 安装 Node.js (推荐 v16+)。
- 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
- 在微信公众平台注册一个小程序，获取 `AppID`。

### 2. 项目配置

1.  修改项目根目录下 `project.config.json` 中的 `appid` 为您自己的 AppID。
2.  进入 `src/App.vue`，将 `uni.cloud.init` 里的 `env` 替换为您自己的云开发环境 ID。

### 3. 安装依赖与运行

```bash
npm install
npm run dev:mp-weixin
```

编译完成后，在微信开发者工具中导入项目根目录下的 `dist/dev/mp-weixin` 目录即可预览。

### 4. 云端部署

在微信开发者工具的“云开发”面板中：

1.  创建所需的数据集合（Collections）：`Users`, `Diaries`, `Comments`, `ActivityLog`。
2.  **设置数据权限**：为了前端能顺利通过云函数读取图片和基础配置，建议将各集合权限设置为“所有用户可读，仅创建者可读写”（因为主要读写都在云函数中跨越权限执行）。
3.  依次右键 `cloudfunctions` 目录下的四个云函数文件夹，选择 **“上传并部署：云端安装依赖(不上传node_modules)”**。

## 💡 性能优化说明

- **按需注入与压缩**：在 `manifest.json` 中配置了 `"lazyCodeLoading": "requiredComponents"` 和代码混淆压缩，大幅降低了小程序的启动时间和内存占用。
- **本地缓存**：充分利用 `uni.setStorageSync`，缓存上一次补传的时间，提升连续补传体验。
- **按需轮询**：全局心跳定时器仅在 `onShow` 期间激活，`onHide` 立即销毁。
